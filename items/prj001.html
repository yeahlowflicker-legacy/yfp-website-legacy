Introduction

This is a simple countdown clock aiming to improve one's productivity with the following features:

  • Auto-looping countdown
  • Configurable time period
  • Toggleable notification sound

The application is built with using the following tools:

  • Python 3.8
  • PySide2
  • Qt/QML
  • PyInstaller
