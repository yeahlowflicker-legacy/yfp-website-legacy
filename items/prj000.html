Introduction

This is a simple text editor which provides basic text editing features. It includes features of:

  • Text Editing
  • Saving/Loading
  • Customizable Theme/Font Size

The application is built with using the following tools:

  • Python 3.8
  • PySide2
  • Qt/QML
  • PyInstaller
